# 🐉 Vorasius
## TL;DR
- 💥 **Primordial Roar** – stack near boss → big raid damage + knockback
- 🌀 **Shadowclaw Slam** – tanks soak smash circles (raid wipes if none hit)
- 🌊 **Aftershock rings** – melee step into slam center, ranged outrange
- 🐛 **Blistercreeps** – drag to walls and kill to break them
- 🔥 **Void Breath** – giant laser → run opposite the casting hand
## Roles
**Tanks**
- 🔁 Tank 1 takes first **2 Shadowclaw Slams** (gets *Smashed*)
- 🔁 Tank 2 handles smashes after that
- ⚠️ If no one soaks a smash → raid damage
**Healers**
- 💥 **Primordial Roar** burst damage
- 📈 Damage ramps from **Primordial Power** stacks over time
**DPS**
- 🐛 Drag **Blistercreeps** to walls (melee left / ranged right)
- 💥 Kill them **on the wall** to break it
- 🌀 Dodge creep spawn swirlies
## Assignments
- 🧱 **Break both walls** with Blistercreep explosions
- 🏃 **Laser:** watch which hand casts → run opposite direction
- 🚀 **Lust:** On pull
